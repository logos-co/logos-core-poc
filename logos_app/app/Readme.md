# Logos Core POC

## Building the Application

### Prerequisites

- Qt 6.x
- CMake 3.16 or higher
- C++17 compatible compiler

### Build Instructions

1. Create a build directory:
   ```
   mkdir -p app/build
   cd app/build
   ```

2. Run CMake:
   ```
   cmake ..
   ```

3. Build the application:
   ```
   make
   ```

4. Run the application:
   ```
   ./LogosApp
   ```
